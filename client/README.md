# Dental Clinic Management System
## Client


## Installation
1. Move to client folder
```bash
cd client
```
2. Run the following command to install the dependencies
```bash
npm install
```
3. Run the following command to start the server
```bash
npm run dev
```
4. Build the project
```bash
npm run build
```
5. Open the following link in your browser
```bash
http://localhost:5173
```
